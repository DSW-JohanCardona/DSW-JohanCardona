Proposito del sitio

Este sitio tiene el proposito de mostrar el uso e implementación de FlexBox y CSS grid los estudiantes que desean aprender sobre diseño de sitios web,
dominar este apartado es fundamental para la creación de sitios web modernos.

Contenidos que se publicarán (Borrador)

Los temas que se pulicarán en el sitio web, estaran destinados a la adquisición de una conocimiento práctico sobre el uso de FlexBox y CSS grid, de css grid se abordarán temas tales como
  • Css Grid, concepto
  • Grid Layout
  • Elementos Grid
  • Columnas
  • Filas
  • Espaciado
  • Alineamiento

Para el contenido relacionado con FlexBox, se encontrarán los siguientes temas:
  • FlexBox, concepto
  • Estructuras básicas
  • Posicionamiento
  • Desing System
